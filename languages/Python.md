# Python

High-level, interpreted language emphasizing readability and simplicity.

## Overview

| Aspect | Details |
|--------|---------|
| Paradigm | Multi-paradigm (OOP, functional, procedural) |
| Typing | Dynamic, strong |
| Interpreted | Yes (CPython, PyPy) |
| First release | 1991 |
| Designed by | Guido van Rossum |

---

## Key Features

- Clean, readable syntax
- Dynamic typing with optional type hints
- Extensive standard library
- Package ecosystem (PyPI)
- Multiple paradigms supported
- REPL for interactive development

---

## Common Use Cases

| Domain | Frameworks/Libraries |
|--------|---------------------|
| Web | Django, Flask, FastAPI |
| ML/AI | TensorFlow, PyTorch, scikit-learn |
| Data Science | pandas, NumPy, Matplotlib |
| Scripting | Built-in, Click, Typer |
| Automation | Selenium, Ansible |

---

## When to Use Python

**Strengths:**

- Rapid development
- Extensive ecosystem
- Great for prototyping
- Dominant in ML/AI

**Considerations:**

- Slower than compiled languages
- GIL limits true parallelism
- Dynamic typing can hide bugs

**Best for:**

- Data science and ML
- Scripting and automation
- Web backends (Django, FastAPI)
- Rapid prototyping
