# C Sharp

Modern, object-oriented language for the .NET ecosystem.

## Overview

| Aspect | Details |
|--------|---------|
| Paradigm | Multi-paradigm (OOP, functional, imperative) |
| Typing | Static, strong |
| Runtime | .NET (CLR) |
| First release | 2000 |
| Designed by | Microsoft (Anders Hejlsberg) |

---

## Key Features

- Strong type system with generics
- LINQ (Language Integrated Query)
- Async/await pattern
- Pattern matching
- Records and init-only properties
- Nullable reference types
- Cross-platform via .NET Core/.NET 5+

## When to Use C#

**Strengths:**

- Mature, feature-rich language
- Excellent tooling (Visual Studio, Rider)
- Strong enterprise adoption
- Unity game development

**Considerations:**

- .NET ecosystem lock-in
- Historically Windows-focused
- Larger runtime than Go/Rust

**Best for:**

- Enterprise applications
- Game development (Unity)
- Windows desktop apps
- Web APIs (ASP.NET Core)
